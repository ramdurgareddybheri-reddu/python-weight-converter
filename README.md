# Python Weight Converter ⚖️

A beginner-friendly Python program that converts weight between **pounds (lbs)** and **kilograms (kgs)** based on user input.  
This project demonstrates the use of:
- User input handling
- Conditional statements
- Arithmetic operations
- Formatted string output

---

## 📝 How It Works
1. The program asks the user to enter their weight.
2. The user specifies whether the input is in **lbs** or **kgs**.
3. The program converts the weight using the factor `0.45`.
   - If input is in lbs → converts to kgs
   - If input is in kgs → converts to lbs
4. The result is displayed with a clear message.
